# PATCH: Ranking / Round 2 / Shoot-off rules

ปรับตามคำสั่งครูรัก:

1. รอบ 1 จัดอันดับแบบต่อเนื่อง 1 ถึงคนสุดท้าย ไม่ให้ระบบปล่อย Class ซ้ำเอง
2. อนุญาตลำดับซ้ำเฉพาะกลุ่มคะแนนรวมเท่าคนสุดท้ายของสิทธิ์ตีรอบ 2 เท่านั้น
3. คนที่ต่ำกว่าเส้นสิทธิ์รอบ 2 แล้ว ไม่ขึ้น Shoot-off
4. ถ้า TOTAL -> 5 -> 3 ยังเท่ากันในช่วงที่มีผลก่อนถึงเส้นรอบ 2 ให้ขึ้น Shoot-off หลังทุกคนในกลุ่มกดจบการตีครบ
5. หน้า Overview รอบ 2 ก่อนมีคนตี เรียงตามคิวตี: คะแนนรอบ 1 ต่ำสุดในกลุ่มรอบ 2 ตีก่อน ไปหาคะแนนดีสุดที่ไม่เข้ารอบตรงตีท้าย
6. เมื่อมีคนเริ่มตี/มีคะแนนรอบ 2 แล้ว หน้า R2 จัดอันดับเฉพาะคนที่ตีแล้วไว้ด้านบน ส่วนคนยังไม่ตีคงเรียงตามคิวเดิม
7. การคัดผู้ผ่านจากรอบ 2 เข้า Bracket ใช้เฉพาะคนที่ตีรอบ 2 จบแล้ว และยังไม่ตั้ง passed_cut จนกว่าผู้มีสิทธิ์รอบ 2 จะตีจบครบ
