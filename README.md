# Petanque Shooting System (Flask)

ระบบบันทึกคะแนนเปตอง Shooting ที่สร้างตามสเปกในแชตนี้ โดยใช้ Python + Flask + SQLite

## ฟีเจอร์หลัก
- สิทธิ์ผู้ใช้ `superadmin / admin / user`
- สร้างอีเวนต์ Shooting พร้อมตั้งค่า
  - รุ่นแข่งขัน
  - ประเภท
  - วันแข่งขัน
  - สถานที่
  - จำนวนสนาม
  - จากรอบแรกเข้ารอบทันที กี่คน
  - มีรอบ 2 หรือไม่
  - ถ้ามีรอบ 2 ตีถึงลำดับที่เท่าไหร่
  - รอบถัดไปคือรอบอะไร
- เพิ่มนักกีฬา, สังกัด, ลำดับยิง, กระจายสนามอัตโนมัติ
- หน้ารวมแบบ scoreboard แยก Atelier 1-5 และ 6m/7m/8m/9m/Total
- หน้ากรอกคะแนนรายคน พร้อมบันทึก/จบการตี/เที่ยวพิเศษ
- ตัดอันดับตามกติกา
  1. คะแนนรวม
  2. จำนวน 5 คะแนน
  3. จำนวน 3 คะแนน
  4. ตีเที่ยวพิเศษ 7 เมตรทุกสถานี
- รองรับการรวมคะแนนรอบแรก + รอบ 2 เพื่อคัดอันดับต่อ
- แดชบอร์ดสถิติ เช่น นักกีฬาคะแนนสูงสุดและสังกัดที่ทำคะแนนดีที่สุด
- ปุ่มบันทึกตารางเป็นรูปภาพ และพิมพ์เอกสาร

## โครงสร้างไฟล์
- `app.py` แอปหลัก
- `templates/` หน้าเว็บทั้งหมด
- `static/scorecard_template.png` ใช้เป็นพื้นหลังใบกรอกคะแนนต้นแบบ
- `instance/shooting.db` ฐานข้อมูล SQLite (สร้างตอนรัน)

## วิธีรัน
```bash
python -m venv .venv
source .venv/bin/activate   # Windows ใช้ .venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

เปิดใช้งานที่ `http://127.0.0.1:8000`

## บัญชีเริ่มต้น
- `superadmin / admin1234`
- `admin / admin1234`
- `viewer / viewer1234`

## หมายเหตุ
- ใบกรอกคะแนนใช้ภาพต้นแบบที่ผู้ใช้อัปโหลดเป็นพื้นหลัง
- การ export เป็นรูปภาพใช้ `html2canvas` ผ่าน CDN ในเบราว์เซอร์
- สามารถต่อยอดเพิ่มลายเซ็นแบบวาดนิ้ว, Socket realtime, หรือ PostgreSQL/Railway ได้ต่อ


## เพิ่มนักกีฬาจาก Excel
- ไปที่หน้าเพิ่มนักกีฬาในอีเวนต์
- อัปโหลดไฟล์ `.xlsx`
- ต้องมีหัวคอลัมน์ `ชื่อ` และ `สังกัด`
- ระบบจะสร้างหมายเลขและลำดับให้อัตโนมัติ
